how to install doxygen:
***********************

install doxygen:
            http://www.stack.nl/~dimitri/doxygen/

or, if you use ubuntu:
            apt-get install doxygen


install graphviz to enable generating diagrams:
            http://www.graphviz.org

or, if you use ubuntu:
            apt-get install graphviz

install mscgen:
            http://www.mcternan.me.uk/mscgen/index.html



how to set up doxygen for your project:
***************************************

on command line:
            cd <your.project.dir>
            doxygen
this will create a template you have to adapt against your porject.

look at the keys in the properties file:
cat character-sheet.doxygen.cfg | grep "^[_A-Z]*\s*=.*"

the properties you have to / should adapt are:

PROJECT_NAME                for example: foo

OUTPUT_DIRECTORY            for example: /home/andre/workspace/foo/target/doxygen

OPTIMIZE_OUTPUT_JAVA        (-:

WARN_LOGFILE                for example: /home/andre/workspace/foo/target/doxygen/warnings.log
                            (note that the doxygen dir must already exist when generating doc!)

INPUT                       for example: /home/andre/workspace/foo/src

EXCLUDE                = \
/home/andre/workspace/foo/src/com/myapp/config/IConstants.java \
/home/andre/workspace/foo/src/com/myapp/view/swing/IActionCommands.java \
/home/andre/workspace/foo/src/com/myapp/thirdparty

MSCGEN_PATH            = /opt/mscgen (where ever you have installed it...)
